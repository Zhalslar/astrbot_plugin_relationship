# 更新日志

## v3.0.3

新功能：

- 引入仅限开发者使用的命令，用于发送好友请求，支持可选的验证信息、备注和回答字段，并支持通过 @ 提及或 QQ 号指定目标。
- 引入仅限开发者使用的命令，用于发送加群请求，支持可选的回答字段，并可从上下文中自动填充。

增强：

- 改进与受限功能相关的提示信息，明确说明某些功能仅对开发者开放且需要额外的源代码支持。

文档：

- 更新 README，记录新的好友与群申请命令、其参数及仅限开发者使用的性质，并说明如何获取所需的闭源模块。

日常维护：

- 添加占位文件 `core/expansion.py`，说明真实实现为闭源并以私下形式分发。

## v3.0.2

在获取群组信息和原始消息时，处理可能出现的 null 和非字典类型响应，以防止在通知处理过程中发生故障。

Bug 修复：

- 当 `get_group_info` 返回 null 时，通过默认使用空字典来保护后续对群组元数据的访问。
- 当原始消息负载不是字典时，跳过通知处理，以避免运行时错误。
- 防止在群组或发送者 ID 不是纯数字时，消息转发发生失败。

Pref 优化：

- 去除“感谢解禁”的废话
